# Nacos_scan
## 简介
Nacos_scan是一款基于python的Nacos服务漏洞扫描工具。

**已集成漏洞**
- Nacos默认密码
- Nacos默认密钥
- Nacos V1 API 未授权访问
- Nacos-Server身份验证绕过
- ServerIdentity鉴权绕过
- Nacos 未授权SQL语句执行
- Nacos 集群反序列化漏洞(版本号判断)
- Nacos Jraft 文件读写漏洞(版本号判断)
